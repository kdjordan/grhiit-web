## Section 3: The Shift

**Headline:**
```
GRHIIT RECALIBRATES WHAT HARD MEANS.
```

**Body:**
```
8 weeks. 24 sessions. 3 per week.

Four movements. Maximum effort. Short intervals. No negotiating with the timer.
```

**Features:**

**1. Bodyweight only**
```
No equipment. No excuses. Your body is the only weight you need to move.
```

**2. Timed intervals**
```
The timer says go. You go. You count your reps. You fight to hold that number as the session tries to break you down.
```

**3. Forward-only progression**
```
No do-overs. No repeats. Each session is unique. Once it's done, it's done. You move forward or you start the cycle over.
```

**4. Progressive intensity**
```
Week 1: 6 seconds work, 3 seconds rest.
Week 8: 12 seconds work, 2 seconds rest.

The sessions don't get easier. They get longer. You become harder to break.
```

**Headline:**

```
8 WEEKS. ONE DIRECTION: FORWARD.
```

**Subheadline:**

**Phase breakdown:**

```
Week 1-2: 6s work / 3s rest
Survival. Your excuses will scream. Keep moving.

Week 3-4: 8s work / 3s rest
Adaptation. Your body begins to obey your will.

Week 5-6: 10s work / 2s rest
Hardening. Discomfort becomes familiar.

Week 7-8: 12s work / 2s rest
Forged. You are not the same person who started.
```

**Closing line:**

```
The sessions don't get easier. You get harder to break.

And that doesn't stay in the workout. It bleeds into everything.
```

## Key Differences from v1

| Element | v1 (Accusatory) | v2 (Aspirational + Edge) |
|---------|-----------------|--------------------------|
| Opening | "You're not training. You're just working out." | "You already know you're capable of more." |
| Problem framing | "Stop lying to yourself" | "Your gauge is broken" (diagnosis, not blame) |
| Edge source | Shame language | Program structure + calibration insight |
| Scarcity | "Not everyone gets in" (fake) | "First 1,000" (real, simple) |
| App description | "It knows when you quit" (surveillance) | Removed - focus on the user, not the tracking |
| Transformation | Week-by-week fitness gains | Week-by-week identity shift + life recalibration |

## Notes

- The calibration insight is now the centerpiece, not buried
- Edge comes from the diagnosis ("your gauge is broken") and the structure ("no negotiating with the timer")
- Opening creates alliance, not confrontation
- Transformation section ties back to the core promise: everything else gets easier because you now know what hard actually is
- Removed fake scarcity and surveillance language
- Tightened feature copy - punchy, not preachy
